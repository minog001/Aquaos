📋 FAULT DIAGNOSIS GUIDE
อาการ สาเหตุที่เป็นไปได้ วิธีแก้ไข ความเร่งด่วน
OFFLINE — ไม่มีข้อมูลเลย ไฟดับ / WiFi ขาด / ESP32 crash ตรวจ power supply, reboot ESP32, ตรวจ AP CRITICAL
STALE DATA — ข้อมูลเก่า >3 นาที Network congestion / loop หยุด / MQTT drop Ping device, ตรวจ loop() ใน firmware, restart HIGH
FROZEN — ค่าไม่เปลี่ยน 5+ รอบ เซ็นเซอร์ขาด / short circuit / I2C timeout ตรวจสาย sensor, ทดสอบ I2C address, เปลี่ยน sensor HIGH
TEMP OUT OF RANGE Heater/chiller เสีย หรือ sensor DS18B20 ผิดพลาด ตรวจ heater, calibrate sensor, เปลี่ยน probe CRITICAL
pH OUT OF RANGE pH probe เก่า/ต้อง calibrate / ค่า electrolyte เปลี่ยน Calibrate pH probe ด้วย buffer 4.0/7.0/10.0, ล้าง probe CRITICAL
